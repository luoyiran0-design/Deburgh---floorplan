De Burgh Floorplate

This is a simple index.html template for GitHub Pages. Put this file in the same folder as your exported PNG floor plate and rename the image file to floorplate.png.

How to use:
1. Save this file as index.html.
2. Put your exported PNG in the same folder and name it floorplate.png.
3. Upload both files to your GitHub repository.
4. Turn on GitHub Pages in repository settings.

Folder structure:
index.html
floorplate.png

How to edit hotspots:
Change href to the real Apply link.
Change data-label to the real unit number or address.
Adjust top, left, width, and height until the box matches the unit on the floor plate.
